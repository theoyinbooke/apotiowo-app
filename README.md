# Apotiowo Landing Page

This is the official landing page and privacy policy for the Apotiowo personal finance management app.

## Live Site

Visit: [https://theoyinbooke.github.io/apotiowo-app/](https://theoyinbooke.github.io/apotiowo-app/)

## Pages

- **Home** - Landing page showcasing app features
- **Privacy Policy** - Full privacy policy for App Store compliance

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS 4
- Lucide Icons
- React Router

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment

The site is automatically deployed to GitHub Pages via GitHub Actions when changes are pushed to the main branch.

## About Apotiowo

Apotiowo is a personal finance management app that helps you:
- Track budgets and expenses
- Set and achieve savings goals
- Manage credit cards
- Analyze spending patterns
- And much more!

Available on the [App Store](https://apps.apple.com/app/apotiowo).
